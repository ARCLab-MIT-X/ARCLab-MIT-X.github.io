BEAVR Sim

GPU-Accelerated RL & Imitation Learning for Robotic Manipulation

MuJoCo + MuJoCo Menagerie + Isaac Lab LeRobot Compatible GPU Acceleration Parallel Environments Task-Based Learning

Key Features

  • Capable of running parallel environments on a single GPU
  • Unified API for observations and actions via LeRobot
  • YAML-based robot configurations, which allow easy use of the entire MuJoCo Menagerie library
  • Flexible task and scene definitions allowing scene building (as seen below)
  • Support for RL and Imitation Learning on any LeRobot-compatible environment
  • 4 pre-built scenes with 100 human demonstrations each collected via teleoperation using BEAVR-teleop
  • Integration with Isaac Lab APIs
  • Fully compatible with LeRobot APIs for training and evaluation

Demonstrations

Below are clips from datasets we collected with the BEAVR-teleop system in MuJoCo simulation designed to test algorithmic physical intelligence. These tasks vary in difficulty in their demand of memory and dexterity.

Pick and Place

Robotic manipulation task demonstrating object grasping and precise placement. The system learns to pick up objects from various positions and place them at target locations, showcasing the effectiveness of our parallel training approach.

Manipulation Grasping RL

Shell Game

Advanced tracking and manipulation task where the robot must follow and interact with moving objects. This demonstrates the system's ability to handle dynamic environments and complex decision-making through imitation learning.

Tracking Dynamic IL

Server Swap - Overhead View

Top-down perspective of a bimanual manipulation task simulating server module replacement. The robot coordinates both arms to remove and insert server components, demonstrating precise dual-arm coordination and task planning capabilities.

Bimanual Coordination Space Ops

Server Swap - Egocentric View

First-person perspective of the server swap task, showing the detailed interaction from the robot's viewpoint. This view highlights the precision required for space robotics operations and maintenance tasks.

Egocentric Precision Assembly

Vanishing Blueprint

Visualization of the scene configuration and task setup process. This shows how the simulation environment is constructed from YAML configurations and MuJoCo assets, providing insight into the flexible scene definition system.
